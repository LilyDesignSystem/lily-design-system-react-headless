# MaltaNationalIdentificationNumberView

A headless read-only display for Malta's Malta National Identification Number. Format: 8 characters: 7 digits and a letter (A, B, G, H, L, M, P, or Z) that encodes geographic origin and registration era.

## What it is

A React 19 component that renders an inline `<span>` for displaying Malta's Malta National Identification Number read-only.

## When to use it

- Displaying a Malta Malta National Identification Number.

## When not to use it

- Use `MaltaNationalIdentificationNumberInput` for the editable input counterpart.
