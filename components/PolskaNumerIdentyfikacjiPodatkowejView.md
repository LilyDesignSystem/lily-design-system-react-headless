# PolskaNumerIdentyfikacjiPodatkowejView

A headless read-only display for Poland's Numer Identyfikacji Podatkowej (NIP). Format: 10 numeric digits used for tax identification.

## What it is

A React 19 component that renders an inline `<span>` for displaying Poland's Numer Identyfikacji Podatkowej (NIP) read-only.

## When to use it

- Displaying a Poland Numer Identyfikacji Podatkowej.

## When not to use it

- Use `PolskaNumerIdentyfikacjiPodatkowejInput` for the editable input counterpart.
