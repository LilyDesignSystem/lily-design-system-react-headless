# ItaliaCodiceFiscaleView

A headless read-only display for Italy's Codice fiscale (CF). Format: 16 alphanumeric characters: 3 from the surname, 3 from the given name, 5 encoding date of birth and sex, 4 encoding place of birth, and 1 check character.

## What it is

A React 19 component that renders an inline `<span>` for displaying Italy's Codice fiscale (CF) read-only.

## When to use it

- Displaying a Italy Codice fiscale.

## When not to use it

- Use `ItaliaCodiceFiscaleInput` for the editable input counterpart.
