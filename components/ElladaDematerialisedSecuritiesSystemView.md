# ElladaDematerialisedSecuritiesSystemView

A headless read-only display for Greece's Dematerialised Securities System (DSS). Format: 10 digits linked to the investor's personal details (name, ID number, passport number, tax registration number) and managed by the Central Securities Depository of Greece.

## What it is

A React 19 component that renders an inline `<span>` for displaying Greece's Dematerialised Securities System (DSS) read-only.

## When to use it

- Displaying a Greece Dematerialised Securities System.

## When not to use it

- Use `ElladaDematerialisedSecuritiesSystemInput` for the editable input counterpart.
