# LiechtensteinNationalIdentityCardNumberView

A headless read-only display for Liechtenstein's Liechtenstein National Identity Card Number. Format: 2 letters followed by 8 digits (e.g. ID022143586); changes with each renewed card.

## What it is

A React 19 component that renders an inline `<span>` for displaying Liechtenstein's Liechtenstein National Identity Card Number read-only.

## When to use it

- Displaying a Liechtenstein Liechtenstein National Identity Card Number.

## When not to use it

- Use `LiechtensteinNationalIdentityCardNumberInput` for the editable input counterpart.
