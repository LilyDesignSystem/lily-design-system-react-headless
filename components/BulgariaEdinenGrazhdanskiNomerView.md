# BulgariaEdinenGrazhdanskiNomerView

A headless read-only display for Bulgaria's Единен граждански номер / Edinen grazhdanski nomer (EGN). Format: 10 digits: the first 6 are the date of birth (YYMMDD), the next 3 encode area and birth order (ninth digit even for boy, odd for girl), and the tenth is a check digit.

## What it is

A React 19 component that renders an inline `<span>` for displaying Bulgaria's Единен граждански номер / Edinen grazhdanski nomer (EGN) read-only.

## When to use it

- Displaying a Bulgaria Единен граждански номер / Edinen grazhdanski nomer.

## When not to use it

- Use `BulgariaEdinenGrazhdanskiNomerInput` for the editable input counterpart.
