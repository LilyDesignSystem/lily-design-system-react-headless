# EnglandNationalHealthServiceNumberView

A headless read-only display for England's National Health Service Number (NHS). Format: 10 digits in 3-3-4 format with a Modulus-11 check digit, allocated through the Personal Demographics Service.

## What it is

A React 19 component that renders an inline `<span>` for displaying England's National Health Service Number (NHS) read-only.

## When to use it

- Displaying a England National Health Service Number.

## When not to use it

- Use `EnglandNationalHealthServiceNumberInput` for the editable input counterpart.
