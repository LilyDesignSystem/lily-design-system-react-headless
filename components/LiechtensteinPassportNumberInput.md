# LiechtensteinPassportNumberInput

A headless input for Liechtenstein's Liechtenstein Passport Number. Format: 1 letter followed by 5 digits (e.g. R00536).

## What it is

A React 19 component that renders a native `<input type="text">` for entering Liechtenstein's Liechtenstein Passport Number.

## When to use it

- Forms collecting a Liechtenstein Liechtenstein Passport Number.

## When not to use it

- Use `LiechtensteinPassportNumberView` for the read-only display counterpart.
