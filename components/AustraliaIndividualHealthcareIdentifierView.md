# AustraliaIndividualHealthcareIdentifierView

A headless read-only display for Australia's Individual Healthcare Identifier (IHI). Format: 16 digits assigned by the Healthcare Identifiers Service.

## What it is

A React 19 component that renders an inline `<span>` for displaying Australia's Individual Healthcare Identifier (IHI) read-only.

## When to use it

- Displaying a Australia Individual Healthcare Identifier.

## When not to use it

- Use `AustraliaIndividualHealthcareIdentifierInput` for the editable input counterpart.
