# HrvatskaOsobniIdentifikacijskiBrojView

A headless read-only display for Croatia's Osobni identifikacijski broj (OIB). Format: 11 digits — 10 random digits plus a check digit; permanent, unique, and assigned to every Croatian citizen and legal person.

## What it is

A React 19 component that renders an inline `<span>` for displaying Croatia's Osobni identifikacijski broj (OIB) read-only.

## When to use it

- Displaying a Croatia Osobni identifikacijski broj.

## When not to use it

- Use `HrvatskaOsobniIdentifikacijskiBrojInput` for the editable input counterpart.
