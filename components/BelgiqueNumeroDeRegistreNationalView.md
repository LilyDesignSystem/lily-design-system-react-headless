# BelgiqueNumeroDeRegistreNationalView

A headless read-only display for Belgium's Numéro de Registre National / Rijksregisternummer (NRN). Format: 11 digits where the first 6 are the date of birth (YYMMDD), the next 3 are an ordering number (uneven for men, even for women) and the last 2 a check digit.

## What it is

A React 19 component that renders an inline `<span>` for displaying Belgium's Numéro de Registre National / Rijksregisternummer (NRN) read-only.

## When to use it

- Displaying a Belgium Numéro de Registre National / Rijksregisternummer.

## When not to use it

- Use `BelgiqueNumeroDeRegistreNationalInput` for the editable input counterpart.
