# PortugalNumeroDeIdentificacaoFiscalView

A headless read-only display for Portugal's Número de Identificação Fiscal (NIF). Format: 9 digits; the leading digit(s) encode the kind of taxpayer (1-3 personal, 45 non-resident citizen, 5 legal person, etc.).

## What it is

A React 19 component that renders an inline `<span>` for displaying Portugal's Número de Identificação Fiscal (NIF) read-only.

## When to use it

- Displaying a Portugal Número de Identificação Fiscal.

## When not to use it

- Use `PortugalNumeroDeIdentificacaoFiscalInput` for the editable input counterpart.
