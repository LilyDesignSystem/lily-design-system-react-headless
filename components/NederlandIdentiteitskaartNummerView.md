# NederlandIdentiteitskaartNummerView

A headless read-only display for Netherlands's Identiteitskaart Nummer. Format: 9 characters following the same format as the passport number; 'O' is disallowed but '0' is permitted.

## What it is

A React 19 component that renders an inline `<span>` for displaying Netherlands's Identiteitskaart Nummer read-only.

## When to use it

- Displaying a Netherlands Identiteitskaart Nummer.

## When not to use it

- Use `NederlandIdentiteitskaartNummerInput` for the editable input counterpart.
