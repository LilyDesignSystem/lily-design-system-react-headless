# CeskoCestovniPasView

A headless read-only display for Czech Republic's Cestovní pas. Format: usually 8 digits, sometimes longer, issued by the Ministry of the Interior.

## What it is

A React 19 component that renders an inline `<span>` for displaying Czech Republic's Cestovní pas read-only.

## When to use it

- Displaying a Czech Republic Cestovní pas.

## When not to use it

- Use `CeskoCestovniPasInput` for the editable input counterpart.
