# SlovenskoRodneCisloView

A headless read-only display for Slovakia's Rodné číslo (RČ). Format: 10 digits in the form YYMMDDCCCX where MM is 01-12 for males and 51-62 for females; X is a check digit and the whole number is divisible by 11.

## What it is

A React 19 component that renders an inline `<span>` for displaying Slovakia's Rodné číslo (RČ) read-only.

## When to use it

- Displaying a Slovakia Rodné číslo.

## When not to use it

- Use `SlovenskoRodneCisloInput` for the editable input counterpart.
