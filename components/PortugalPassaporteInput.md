# PortugalPassaporteInput

A headless input for Portugal's Passaporte. Format: 1 letter [A-Z] and 6 digits [0-9].

## What it is

A React 19 component that renders a native `<input type="text">` for entering Portugal's Passaporte.

## When to use it

- Forms collecting a Portugal Passaporte.

## When not to use it

- Use `PortugalPassaporteView` for the read-only display counterpart.
