# AustraliaIndividualHealthcareIdentifierInput

A headless input for Australia's Individual Healthcare Identifier (IHI). Format: 16 digits assigned by the Healthcare Identifiers Service.

## What it is

A React 19 component that renders a native `<input type="text">` for entering Australia's Individual Healthcare Identifier (IHI).

## When to use it

- Forms collecting a Australia Individual Healthcare Identifier.

## When not to use it

- Use `AustraliaIndividualHealthcareIdentifierView` for the read-only display counterpart.
