# RomaniaPasaportView

A headless read-only display for Romania's Paşaport. Format: 8 characters: positions 1-2 [A-Z] and positions 3-8 [0-9].

## What it is

A React 19 component that renders an inline `<span>` for displaying Romania's Paşaport read-only.

## When to use it

- Displaying a Romania Paşaport.

## When not to use it

- Use `RomaniaPasaportInput` for the editable input counterpart.
