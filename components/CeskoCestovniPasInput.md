# CeskoCestovniPasInput

A headless input for Czech Republic's Cestovní pas. Format: usually 8 digits, sometimes longer, issued by the Ministry of the Interior.

## What it is

A React 19 component that renders a native `<input type="text">` for entering Czech Republic's Cestovní pas.

## When to use it

- Forms collecting a Czech Republic Cestovní pas.

## When not to use it

- Use `CeskoCestovniPasView` for the read-only display counterpart.
