# DeutschlandKrankenversichertennummerView

A headless read-only display for Germany's Krankenversichertennummer (KVNR). Format: a random capital letter followed by eight random digits and a Luhn check digit.

## What it is

A React 19 component that renders an inline `<span>` for displaying Germany's Krankenversichertennummer (KVNR) read-only.

## When to use it

- Displaying a Germany Krankenversichertennummer.

## When not to use it

- Use `DeutschlandKrankenversichertennummerInput` for the editable input counterpart.
