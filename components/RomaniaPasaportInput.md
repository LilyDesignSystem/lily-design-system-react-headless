# RomaniaPasaportInput

A headless input for Romania's Paşaport. Format: 8 characters: positions 1-2 [A-Z] and positions 3-8 [0-9].

## What it is

A React 19 component that renders a native `<input type="text">` for entering Romania's Paşaport.

## When to use it

- Forms collecting a Romania Paşaport.

## When not to use it

- Use `RomaniaPasaportView` for the read-only display counterpart.
