# SlovenskoPasInput

A headless input for Slovakia's Pas. Format: 9 characters in the format XXNNNNNNN: 2 block letters and 7 digits; valid for 10 years.

## What it is

A React 19 component that renders a native `<input type="text">` for entering Slovakia's Pas.

## When to use it

- Forms collecting a Slovakia Pas.

## When not to use it

- Use `SlovenskoPasView` for the read-only display counterpart.
