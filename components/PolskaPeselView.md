# PolskaPeselView

A headless read-only display for Poland's PESEL. Format: 11 numeric digits; assigned shortly after birth and unchanged for life.

## What it is

A React 19 component that renders an inline `<span>` for displaying Poland's PESEL read-only.

## When to use it

- Displaying a Poland PESEL.

## When not to use it

- Use `PolskaPeselInput` for the editable input counterpart.
