# NorgeFodselsnummerView

A headless read-only display for Norway's Fødselsnummer. Format: 11 digits where the first 6 represent the date of birth (DDMMYY).

## What it is

A React 19 component that renders an inline `<span>` for displaying Norway's Fødselsnummer read-only.

## When to use it

- Displaying a Norway Fødselsnummer.

## When not to use it

- Use `NorgeFodselsnummerInput` for the editable input counterpart.
