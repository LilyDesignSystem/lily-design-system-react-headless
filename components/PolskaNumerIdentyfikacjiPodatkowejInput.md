# PolskaNumerIdentyfikacjiPodatkowejInput

A headless input for Poland's Numer Identyfikacji Podatkowej (NIP). Format: 10 numeric digits used for tax identification.

## What it is

A React 19 component that renders a native `<input type="text">` for entering Poland's Numer Identyfikacji Podatkowej (NIP).

## When to use it

- Forms collecting a Poland Numer Identyfikacji Podatkowej.

## When not to use it

- Use `PolskaNumerIdentyfikacjiPodatkowejView` for the read-only display counterpart.
