# IslandKennitalaInput

A headless input for Iceland's Kennitala. Format: 10 digits where the first 6 are the date of birth (DDMMYY).

## What it is

A React 19 component that renders a native `<input type="text">` for entering Iceland's Kennitala.

## When to use it

- Forms collecting a Iceland Kennitala.

## When not to use it

- Use `IslandKennitalaView` for the read-only display counterpart.
