# SlovenskoPasView

A headless read-only display for Slovakia's Pas. Format: 9 characters in the format XXNNNNNNN: 2 block letters and 7 digits; valid for 10 years.

## What it is

A React 19 component that renders an inline `<span>` for displaying Slovakia's Pas read-only.

## When to use it

- Displaying a Slovakia Pas.

## When not to use it

- Use `SlovenskoPasInput` for the editable input counterpart.
