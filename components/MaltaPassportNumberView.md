# MaltaPassportNumberView

A headless read-only display for Malta's Malta Passport Number. Format: 7 numerical digits issued by the Civil Registration Directorate.

## What it is

A React 19 component that renders an inline `<span>` for displaying Malta's Malta Passport Number read-only.

## When to use it

- Displaying a Malta Malta Passport Number.

## When not to use it

- Use `MaltaPassportNumberInput` for the editable input counterpart.
