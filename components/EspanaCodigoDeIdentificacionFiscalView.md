# EspanaCodigoDeIdentificacionFiscalView

A headless read-only display for Spain's Código de Identificación Fiscal (CIF). Format: 9 characters: 8 numbers and a control letter (the letters I, Ñ, O, and U are not used).

## What it is

A React 19 component that renders an inline `<span>` for displaying Spain's Código de Identificación Fiscal (CIF) read-only.

## When to use it

- Displaying a Spain Código de Identificación Fiscal.

## When not to use it

- Use `EspanaCodigoDeIdentificacionFiscalInput` for the editable input counterpart.
