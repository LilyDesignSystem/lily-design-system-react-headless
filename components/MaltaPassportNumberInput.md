# MaltaPassportNumberInput

A headless input for Malta's Malta Passport Number. Format: 7 numerical digits issued by the Civil Registration Directorate.

## What it is

A React 19 component that renders a native `<input type="text">` for entering Malta's Malta Passport Number.

## When to use it

- Forms collecting a Malta Malta Passport Number.

## When not to use it

- Use `MaltaPassportNumberView` for the read-only display counterpart.
