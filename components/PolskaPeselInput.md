# PolskaPeselInput

A headless input for Poland's PESEL. Format: 11 numeric digits; assigned shortly after birth and unchanged for life.

## What it is

A React 19 component that renders a native `<input type="text">` for entering Poland's PESEL.

## When to use it

- Forms collecting a Poland PESEL.

## When not to use it

- Use `PolskaPeselView` for the read-only display counterpart.
