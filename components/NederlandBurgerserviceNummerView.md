# NederlandBurgerserviceNummerView

A headless read-only display for Netherlands's Burgerservicenummer (BSN). Format: 9-digit number issued via the Personal Records Database (BRP); validated with a variant of the eleven-test (Luhn) algorithm.

## What it is

A React 19 component that renders an inline `<span>` for displaying Netherlands's Burgerservicenummer (BSN) read-only.

## When to use it

- Displaying a Netherlands Burgerservicenummer.

## When not to use it

- Use `NederlandBurgerserviceNummerInput` for the editable input counterpart.
