# IslandKennitalaView

A headless read-only display for Iceland's Kennitala. Format: 10 digits where the first 6 are the date of birth (DDMMYY).

## What it is

A React 19 component that renders an inline `<span>` for displaying Iceland's Kennitala read-only.

## When to use it

- Displaying a Iceland Kennitala.

## When not to use it

- Use `IslandKennitalaInput` for the editable input counterpart.
