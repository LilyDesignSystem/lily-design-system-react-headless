# LietuvaPasasInput

A headless input for Lithuania's Pasas. Format: 8-digit passport or identity card number.

## What it is

A React 19 component that renders a native `<input type="text">` for entering Lithuania's Pasas.

## When to use it

- Forms collecting a Lithuania Pasas.

## When not to use it

- Use `LietuvaPasasView` for the read-only display counterpart.
