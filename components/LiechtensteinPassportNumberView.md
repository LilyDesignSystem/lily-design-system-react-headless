# LiechtensteinPassportNumberView

A headless read-only display for Liechtenstein's Liechtenstein Passport Number. Format: 1 letter followed by 5 digits (e.g. R00536).

## What it is

A React 19 component that renders an inline `<span>` for displaying Liechtenstein's Liechtenstein Passport Number read-only.

## When to use it

- Displaying a Liechtenstein Liechtenstein Passport Number.

## When not to use it

- Use `LiechtensteinPassportNumberInput` for the editable input counterpart.
