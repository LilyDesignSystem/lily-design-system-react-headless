# LietuvaPasasView

A headless read-only display for Lithuania's Pasas. Format: 8-digit passport or identity card number.

## What it is

A React 19 component that renders an inline `<span>` for displaying Lithuania's Pasas read-only.

## When to use it

- Displaying a Lithuania Pasas.

## When not to use it

- Use `LietuvaPasasInput` for the editable input counterpart.
