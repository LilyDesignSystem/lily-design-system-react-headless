# PortugalPassaporteView

A headless read-only display for Portugal's Passaporte. Format: 1 letter [A-Z] and 6 digits [0-9].

## What it is

A React 19 component that renders an inline `<span>` for displaying Portugal's Passaporte read-only.

## When to use it

- Displaying a Portugal Passaporte.

## When not to use it

- Use `PortugalPassaporteInput` for the editable input counterpart.
